Python: 파이썬은 요즘 웹에서 주목받는 언어다. 하지만 파이썬은 주로 웹에서만 사용되고, 자바와 다르게 콘솔어플을 만들거나 다른 류의 어플을 만들때는 사용을 많이는 안합다. 콘솔어플도 만들수있겠지만, 주로 웹에서만 사용된다. 또한 최근 보면 구글엡엔진에서도 파이썬과 자바만 지원한다. 초보가 배우기 쉽고, c에 비해 개발시간을 단축시킬수있다. 다른 프로그램의 연동이 용의하고 라이브러리가 많다.      

PHP: PHP는 좋은게 HTML과 봐로 연동이 된다. .HTML 파일에 &lt;? <? 사인을 사용한후 php 키워드와 그사이 php코드를 작성한후 ?&gt;   ?> 로 마무리하여 .html 파일속에서 바로 php가 작성되게 할수있다. php의 장점은 html속에 php코드를 이곳저곳에 살짝살짝 끼어넣어서 코드를 흐트러놓을수가 있다. 빠르고 내장된 코어 라이브러리를 가지고 있어서 코딩이 가볍다. 몇가지 기능은 다른 프로젝트에서 사용된 동일한 코드를 재수정하여 사용가능하다. 또 그많큼 많이 사용되고 있으므로 근본적으로 강하고 안정적이다.

Javascript: 자바는 웹, 모바일 등등에 사용된다. 웹에서는 자바를 이용하여 서버를 작성하고 또한 자바스크립을 이용하여 클라이언트를 작성할수있다. 객체지향 언어이기 때문에 프로그램하기가 쉽다. 미리 설치되지 않은 프로그램이 서버로 동적으로 다운되어 웹에서 실행 할 수있다. 멀티스레드가 지원되고, 메모리 관리없이 가상머신이 관리를 대신 해준다. 플랫폼이 독립적이여서 특정 컴퓨터에 종속되지 않는다.

Ruby: 빠른 속도와 배우기 쉬운 장점. 간결한 코딩으로 프로그램을 쉽게 만들수있다. 자바나 닷넷프레임워크처럼 여러가지 설정 코드를 작성해야 하지만, 루비는 작명 규칙을 따르기만 하면 연결을 자동으로 설정해줘서 코드생산 비용을 줄일수있다. 가비지 콜렉션이 있어서 메모리를 적절히 관리, 해제한다. 메모리관리에 고민하지 않아도된다.
