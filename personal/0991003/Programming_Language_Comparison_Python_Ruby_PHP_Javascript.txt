파이선의 장점은 
1.코드를 짜다보면 눈에 딱딱 보여서 입문자도 배우기 쉬운거 같습니다. 
2.또한 코드자체가 많이 압축할수 있는 장점이 있는것 같습니다 씨언어에 비하면 훨씬 많이 압축이 가능할것같습니다.
3.라이브러리가 다양한 것도 장점인것 같습니다 시간이 많지않아 몇개해보지는 못했으나 찾아보니 많았던것 같습니다.
4.유지보수가 쉬운거 같습니다. 찾아보니 메모리를 자동관리해준다고 합니다.
5.여러 다른 프로그램과의 연동이 용의합니다.

다른것과 비교하자면 
씨언어 같은 것보다는 속도가느린점이 단점인거 같습니다.

루비의 장점은
1.순수 객체지향을 지원한다는 것 입니다.
2. 속도가 빠르고 배우기가 쉽다는것 입니다.
3.루비온레일즈라는 프로그램을 통해 를 통해 신속하게 웹어플을 개발가능합니다.
4.DRY(Don't Reapeat Yourself)원칙을 고수하여 코드의 중복을 줄였습니다.
5.관례에 따른 설계를 하면 자동으로 다음것을 설계해줍니다.

단점은
라이브러리도 파이썬에비하면 적습니다.

자바스크립트의 장점은
1.이름이 비슷한 자바에 비하면 훨씬 배우기 쉽다는 장점이 있습니다.
2.플랫폼(운영체제나 하드웨어)에 의존하지 않고 작동합니다.
3.사용자 컴퓨터상에 작동하므로 서버에 주는 부담이 매우 적습니다.
4.클라이언트와 실시간으로 대응할수있다는것 입니다
5.여러 분야에서 사용이 가능합니다.

단점은

1.소스의 노출에 따른 보안입니다.
2.브라우저별 객체나함수의 사용이 달라 호환성이 문제가 됩니다.
3.브라우저에 독립적이지 못합니다.

PHP의 장점은
1.단순한 반복되어지는 작업들을 쉽게 구현할수 있습니다.
2. 날자계산등에 의해 동적인 홈페이지를 구현할수 있습니다.
3.모든 운영체제에서 작동합니다.(플랫폼등에 구애받지 않습니다)
4.데이터베이스를 지원하고 데이터베이스와의 연결하는 작업이 쉽고 빠릅니다.
5.배우기에 아주 쉽고 사용하기에 편리함.
