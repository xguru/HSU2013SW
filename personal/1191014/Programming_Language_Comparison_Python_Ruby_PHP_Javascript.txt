


Python 

1991년 프로그래머 귀도 반 로썸(Guido van Rossum)이 만들었으며, 구글에서 3개 공식언어 (C,Java, Python) 에서 채택할 정도로 요즘 인정받고 있는 프로그래밍 언어이다.

장점

1. 들여쓰기(Indentation)가 의무화가 되어 있어서 비교적 알아보기가 쉽다.
2. 다른 프로그래밍 언어에 비해 개발하는 시간이 짧고 직관적인 편이다.
3. 라이브러리가 매우 광범위하고 확장성이 넓게 퍼져있다.
4. 문자열이 모두 유니코드로 구성되있어서 한글, 한자 등의 문자 표현이 별도의 작업을 하지 않아도 된다.
5. 런타임 상황시 타입 체크를 할 수 있다.








Ruby

유키히로 마츠모토가 만들었으며, 프로그래밍 언어인 펄, 스몰토크, 리스프, 에이다, 아이펠을 융합하여 명령형 프로그래밍언어를 만들었다. 프로그래밍 언어 사용 순위를 제공하는 TIOBE 인덱스에 따르면 10번째로 많이 쓰이는 언어이다. 그리고 2006년에는 '올해의 언어'로 인정받았다.

장점

1. 사용, 수정, 복사, 배포까지도 모든 것이 무료이다.
2. 모든 것이 객체이다. 자신만의 속성과 액션을 갖는다.
3. 매우 유연하기 때문에 코어 부분도 제거하고 재정의할 수 있다.
4. Closer 즉, 블럭이 있는데, 이것을 추가함으로써 메서드의 동작을 기술할 수 있다.
5. 단일 상속만을 제공하는데, 다중 상속보다 훨씬 간결한 방법이다.





 


PHP

HTML에 포함되어있는 스크립팅 언어이다. C, Java, 펄등 다른 프로그래밍 언어에서 많은 문장 형식을 준용하고 있기 때문에 빠르고 쉽게 작성할 수 있다. 또한 명령 줄 인터페이스 방식의 자체 인터프리트를 제공하기 때문에, 범용 프로그래밍 언어로도 사용할 수 있으며, 그래픽 어플리케이션도 제작이 가능하다.

장점

1. HTML 처리에 강점을 가지고 있다.
2. 다양한 데이터베이스를 지원하기 때문에 데이터베이스와 사용자간의 중계 역할을 잘 수행한다.
3. PHP 사용 허가서의 규정따라 릴리즈된 자유 소프트웨어이다.
4. 다양한 라이브러리를 제공하며 다양한 표준을 지원한다.
5. 성능은 업그레이드를 통해 계속해서 향상되고 있다.






Javascript

브라우저에서 실행되는 동적인 언어이다. 웹페이지 안에서 여러 동적인 움직임을 만들어 줄 수 있는 언어이고, HTML/CSS을 동적으로 제어하는 DOM으로 나뉜다.


장점

1. HTML 요소와 그 값을 동적으로 편집이 가능하다
2. 다음에 사용할 때를 대비하여 사용자 컴퓨터에 데이터를 저장하고 검색하기 위한 쿠키를 작성한다.
3. 다양한 플랫폼과 브라우저에서 사용이 가능하다.
4. 다양한 라이브러리를 가지고 있다.
5. 서버와 통시없이 독립적으로 실행이 가능하다.
