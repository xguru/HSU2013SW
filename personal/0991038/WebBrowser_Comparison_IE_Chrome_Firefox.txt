-Internet Explorer-

소감: Chrome 이나 Firefox를 사용한 후 Internet Explorer 써보면 느리다는 것을 확실히 느낄 수 있다.
      MS컴퓨터의 보급으로 인해 Internet Explorer에 익숙하고,그만큼 사용하는데 부담이 없다.

장점: 전세계적으로 예전부터 많이 사용했고,액티브 X 기술의 지원때문에 금융권 업무가 가능하고,
      window media player가 필요한 싸이트의 경우 동영상 재생이 가능하다.(Chrome,Firefox는 재생 불가능)

차이점: Chrome이나 Firefox보다 느린감이 있고, UI가 조금씩 차이가 나지만 우리에게 익숙하다.
        Internet Explorer 는 즐겨찾기라는 용어를 쓰고, Chrome과 Firefox는 북마크라는 용어를 쓴다.

-Chrome-

소감: 우리에게 익숙한 Internet Explorer와 비슷한 기능들이 많아 금방 익숙해졌고, Internet Explorer보다 속도가 빠르다.
      번역기를 이용해 홈페이지의 언어를 번역할 수 있는 창이 기본적으로 나타나는데 100% 완벽하지는 않았다.

장점: Chrome Web Store에 들어가면 여러 가지 앱을 Chrome에 추가 할수 있고 별도의 다운로드 없이 웹에서 앱을 즐길 수 있다.

차이점: Chrome은 Chrome Web Store을 이용해 앱을 웹에서 즐길 수 있다.


-Firefox-

소감: Firefox라는 메뉴버튼이 있어 문서작업 소프트웨어에 인터넷기능을 섞은 느낌이었다.

장점: 주소창 옆에 검색엔진을 등록하고 퀵으로 검색하는 기능이 있어 정보를 검색하는데 간편하고 편리하다. 

차이점: Firefox에서는 Mozilla 한국 커뮤니티 싸이트를 운영하면서 정보를 공유하는데, Twitter나 Facebook까지
	연동이 되기때문에 활동이 활발하다.
	5개의 탭이 열려있다고 가정할때,Internet Explorer나 Chrome는 현재탭을 닫을 것인지 모든탭을 닫을 것인지 묻지만
	Firefox는 현재탭을 닫을 것인지 5개의탭을 모두 닫을 것인지 묻는다. 즉 Firefox는 열려있는 탭의 갯수를 계산한다.
