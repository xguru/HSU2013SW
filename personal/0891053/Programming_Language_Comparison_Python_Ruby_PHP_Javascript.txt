1. Python 언어의 장점

(1) 컴퓨터 언어 임에도 불구하고 사람이라 생각하는 언어이다.
프로그래밍이란 컴퓨터에 인간이 생각하는 것을 입력시키는 행위라고 할 수 있다. 

(2) 강력한 언어이다.
파이썬으로 프로그래머는 대부분의 모든 일들을 해낼 수가 있다.또한 파이썬은  파이썬과 C는 찰떡궁합이란 말이 있다. 


(3)무료이다
파이썬은 오픈소스이므로 당연히 무료이다. 언제 어디서든 파이썬을 다운로드하여 사용할 수 있고, 사용료를 지불해야 할 필요가 없다. (그렇다고 자신이 만든 프로그램을 무조건 공개해야 하는 것은 아니다.)


(4)간결하다
파이썬은 간결하다. 이 간결함은 파이썬을 만든 귀도(Guido)의 의도적인 산물이다. 만약 어떤 언어가 하나의 일을 하기 위한 방법이 100가지라면 파이썬은 가장 좋은 방법 1가지를 선호한다. 

(5) 프로그래밍이 재밌다
 파이썬은 다른 것에 신경 쓸 필요 없이 내가 하고자 하는 부분에만 집중할 수 있게 해주기 때문이다. 





2. Ruby 언어의 장점

(1) 런타임 시 모든 종류의 트릭이 가능하다.
인터프리디드언어고 동적 유형화가 가능하기 때문에 트릭이 가능하다.

(2) 도메인 스팩의 언어를 구현하여서 신택스에서 벗어나 고급 추상화 레벨에서 작업할 수 있다.

(3) 자바가 중요하다고 생각하는 사람들도 Ruby를 많이 사용한다.
(4) 가장 큰 장점은 좋은 개발환경이다. RDT가 모든걸 가능하게 만들었다.






3. PHP 언어의 장점

(1) 서버안에서 해석이 가능한 스크립트 언어다.
(2) 데이터베이스와 connect를 쉽게 해준다.
(3) UNIX와 Windows 환경 모두에서 가능하다.
(4) 코드 작성이 c언어와 비슷하며 class를 지원하여 효율적이면서 간단하다.
(5) 속도가 가장 빠르다는 C언어 견주어도 비슷하다.






4. javascript의 장점
(1) 서버에 부담을 줄일수 있고 웹에 접근하기 위한 언어로 사용되기 좋다.
(2) javascript는 플랫폼에 굉장히 독립적이다.
(3) 일단 java보다 프로그래밍이 쉽고 매우 간단하기 때문에 배우기 쉽다.
(4) 객체지향 언어 이기 때문에 재사용이 가능하다.
(5) 초보자 분들이 가장 하기 쉬운 언어이다.
