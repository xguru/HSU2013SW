Python- 파이썬은 요즘 웹에서 주목받는 언어이다.
        하지만 파이썬은 주로 웹에서만 사용되지 자바와 다르게 콘솔어플을 만들거나
        다른 류의 어플을 만들때는 사용을 많이는 안한다.
        물론 콘솔어플도 만들수있다. 하지만 주로 웹에서만 사용된다.
        파이썬은 C, C++, Java에 비해 평균적으로 삼분의 일 이하의 개발 시간과,
        코드의 라인 수도 C, C++, Java에 비해 대략 삼분의 일이고, Java의 절반 정도의 메모리 소모에,
        전체 수행 속도는 C나 C++보다는 느리지만 Java보다 약간 빠른 정도를 보여준다.
        (주목할 만한 사실은 스크립트 언어 중에서도 파이썬이 높은 성능을 보여줬다는 점이다)
        여기서 주목할 점은 바로 파이썬의 높은 생산성이다.
        같은 시간 동안에 더 많은 코드를 만들 수 있으면서 동시에 전체 코드 길이는 짧다는 점은 파이썬의 대표적인 매력 중 하나이다.
        

Ruby -  ruby는 일본의 마츠모토 유키히로가 개발한 동적 객체 지향 스크립트 프로그래밍 언어이다.
        루비 역시 사용하기 쉬운 언어이며, 인간을 중심으로 설계되어야 한다는 이념에 개발된 언어라고 한다.
        루비는 unix 나 맥 os, 윈도우,도스 등에서 동작 가능하고 자바 환경을 실행시킬 수 있는 어떠한 곳에서도 
        실행 가능하다고한다. 또한 문법이 단순하며 C언어에 의한 확장또한 무난하다고한다.
        또한 Python은 정형화된 들여쓰기를 요구하는 반면 Ruby는 정형화된 서식을 요구하지않아 사용이 더 편리하다.


PHP -   PHP는 스크립트 언어입니다.자바 스크립트와 비슷하다고 생각할수 있습니다. 
        html 문서안에 직접 코딩할수가 있고 PHP로 만든 프로그램의 확장자를 아파치 서버의 설정에 따라서 html으로 사용할 수가 있습니다.  
        그리고 이전에 많은 사용자수를 가지고 있는 Perl 에 비해서 속도가 대단히 빠르다는 장점이 있습니다. 
        (새로 나올 PHP4, Zend 같은 경우에는 인터프리터 형식이 나닌 컴파일을 해서 실행을 하기 때문에 C의 최대 장점이있던
        속도에서도 별로 뒤지지 않는다고 합니다) 또한 문법두 포인터 없는 C와 비슷하다구 생각이 들 정도로 쉽습니다. 
        또 하나, PHP에서 가장 좋은 점이라구도 할수 있는 DB와의 연동이 매우 쉽다는 겁니다. 
        C나 Perl에서두 당근 DB와 연동해서 쓸수 있습니다. 
        그런데 그 사용법이 PHP에 비해서는 아주 어렵습니다
	

Javascript - 자바스크립트는 객체기반 언어이며 자바언어의 기본 문법을 대부분 사용하여 유사하지만 자바보다 배우기 쉬운 언어이다.
             클라이언트에서 실행되기 때문에 네트워크를 통해 서버로 전송하기 않고도 사용자가 입력한 내용을 즉시 확인하여
             처리가 가능하다. 자바스크립트 프로그램 코드는 따로 파일로 작성하지 않고도 웹에서 수행할 HTML 문서내에서
             직접 코딩할 수 있다. 또한 문법이 쉽고 내장 함수 및 객체를 사용하여 프로그래밍이 용이하다.
             Python과 비교했을 때 javascript 둘다 객체 기반이지만 Python은 클래스안에 정의하지 않아도 되는 단순한 함수와 변수를 사용하는 
             프로그래밍 스타일을 지원한다.
