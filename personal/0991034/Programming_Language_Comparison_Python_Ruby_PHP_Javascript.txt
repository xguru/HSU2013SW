Python
처음 파이썬을 실습할때는 조금 당황스러웠습니다. 어떻게 변수형 타입을 정해주지도
않았는데 int, float, string등 알아서 쓸수있는지 의아했는데 두세번 연습해보니
편리함뿐만 아니라 좀더 직관적으로 코딩을 할수 있었습니다.
또한 배열의 메모리할당을 먼저 잡아주지 않아도 되는점 배열에 데이터를 추가시킬때도
간단한 메소드 하나로 해결되는 모습을 보며 놀라웠습니다
c를 기반으로한 언어라 그런지 문법자체가 어렵진 않았습니다.
현재 자바도 배우고 있는데 또 문법내용이 자바랑 유사하거나 똑같은 경우도 있어서
편하게 읽고 쓸수도 있었습니다.
인터넷 조사를 해 추가적인 장점을 찾아보니 기본적으로 제공하는 메소드나
자료형이 많아서 개발속도면에서는 뛰어나다고 합니다 하지만 C와 비교해보면
속도가 느리다고 합니다. 

Ruby
루비같은 경우에는 교수님께서 추천해주신 싸이트를 가봐서 예제를 통해 실습하려고
해봤습니다. 하지만 문법자체가 처음보는 형태라 어려움을 겪었습니다. 이러한류의
언어는 처음 접해봐 전 파이썬보다는 어렵게 실습해보았습니다. 하지만 문법자체가
직관적인 단어로 되어있어 조금만 연습한다면 충분히 습득할수 있는 수준이었습니다
인터넷 조사를통해 추가적인 장점을 찾아보니 순수 객체지향을 지원한다고 합니다
위의 파이썬과는 대조적인 목적을 갖고있었습니다. 또한 빠른 웹
어플리케이션을 개발할수 있다고 합니다.

PHP
PHP는 정확히 HTML언어를 배워본적이 없어서 모르겠지만
쓰는 형식이 매우 비슷해 보였습니다. 몇번을 실습을통해 연습해보니
안드로이드의 xml을 쓰는방식과 많이 유사하였다 하지만
이형식은 웹을 구현하는 언어라 완전 다른 명령어로 웹 페이지를 만드는 방식이었습니다.
몇번정도 연습해보니 손에익는걸 보며 나름 투자하는 시간에 비해 배우기는
쉬울것으로 판단되었습니다.
인터넷에서 정보를 몇가지 더 찾아보니 PHP로 개발을 하면 개발속도가 빠르고
실행속도가 빠르다고 합니다. 그리고 대부분의 데이터베이스를 지원하여 호완성또한 뛰어난다고 합니다

Javascript
자바스크립트는 Java와 이름이 비슷해보여 친숙한 이미지로 연습해보았다.
자바스크립트 실습에 앞서 파이썬을 해봐서인지 파이썬과 유사한 형태로 기본예제를
수행해 보았다.변수선언또한 파이썬과 유사하게 타입을 정해주지 않아도
알아서 자바스크립트가 정해주는데  앞에 var만 붙여주면 됩니다.
구문이 어렵지 않아 C나 java에비해 좀더 쉽게 배울수 있을것 같습니다.
인터넷 정버를 몇가지 더 추가해보면 자바는 객체지향 언어로써 코드의 재사용성이 있으며
사용자컴퓨터에서 작동해 서버에주는부담이 적다고합니다.
또한 가장 강력한점은 바로 플렛폼에 의존하지 않고 구동가능하기때문에
익스플로러처럼 자바스크립트를 지원하는 브라우저만 있다면 어디서든 사용가능하다고 합니다

만약 제가 처음으로 웹언어를 배우게된다면 저에게 제일 익숙했던 파이썬이 1순위가 될것이고
2번째로는 자바스크립트 3번째는 PHP 4번째는 Ruby순으로 배울거같습니다.
